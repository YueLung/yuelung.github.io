建立一個新專案
```
ng new my-project
```
<br>
建置並啟動伺服器
```
ng serve
ng serve --open  
```      
<br>
新增元件(component)
```
ng g c
ng g c --flat		表示不新增資料夾	
ng g c --skip-import  不將此元件新增到其他NgModule 
```
<br>
新增模組(module)
```
ng g m
ng g c --flat
ng g c --routing      新增routing module
```
