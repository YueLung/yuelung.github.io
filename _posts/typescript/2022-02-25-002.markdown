## Promise
<br>
基本用法
```javascript
const promise1 = new Promise((resolve, reject) => {
  setTimeout(() => {
    var rnd = Math.floor(Math.random() * 2); // 0 or 1
    if (rnd == 1) resolve('true');
    else reject('false');
  }, 1000);
});

promise1.then((value) => {
  console.log(value);
})
.catch((value) => {
  console.log(value);
});

console.log("test");

// output
test
// after 1000ms
true or false
```
<br>
串接promise，避免波動拳
```javascript
new Promise((resolve, reject) => {
  setTimeout(() => {
    resolve(1)
  }, 1000);
})
.then((value) => {
  console.log(value); 
  return new Promise((resolve, reject) => {
    setTimeout(()=>{
    resolve(2);  
  },1000);
  }) 
})
.then((value) => {
  console.log(value); 
})

// output
// after 1000ms
1
// after 1000ms
2
```
