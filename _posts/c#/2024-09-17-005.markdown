### boxing
```c#
int i = 123;
// The following line boxes i.
object o = i;
``` 
![alt text](/public/img/c#/005-boxing-1.png)
    
### unboxing
```c#
int i = 123;      // a value type
object o = i;     // boxing
int j = (int)o;   // unboxing
```
